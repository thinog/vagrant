# Vagrant
Aplicação de conteúdo estudado em curso [Vagrant: Gerenciando máquinas virtuais][vagrant-course], da Alura.

Comandos utilizados:

```vagrant init <box>```

```vagrant up```

```vagrant halt```

```vagrant status```

```vagrant destroy```

```vagrant reload```

```vagrant ssh```

```vagrant ssh-config```

[vagrant-course]: https://www.alura.com.br/curso-online-vagrant-gerenciando-maquinas-virtuais
